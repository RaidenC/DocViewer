DocViewer - Header & Filters

How should the search bar and filters be organized?

A

Integrated Search Bar

Single search input with dropdown filters beside it. Channel filter as checkbox group. Date and client filters in expandable "More filters" section.

B

Filter Panel

Full-width filter bar with distinct sections: search on left, channel pills/tags, date range pickers, client dropdown. Always visible.

C

Minimal Header

Just a search bar. All filters hidden in a slide-out panel or dropdown menu. Keeps header clean, requires more clicks to filter.
